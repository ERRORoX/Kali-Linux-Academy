# Системные команды Kali Linux

## Информация о системе

uname -a — показать полную информацию о системе
uname -r — показать версию ядра
uptime — показать время работы системы и загрузку
hostname — показать имя хоста
hostname -I — показать IP-адрес
last reboot — показать историю перезагрузок
date — показать текущие дату и время
cal — показать календарь
w — показать кто в системе
whoami — показать текущего пользователя

## Управление процессами

ps aux — список всех процессов
ps -ef — альтернативный формат списка процессов
top — интерактивный мониторинг процессов
htop — улучшенная версия top
kill PID — завершить процесс по ID
killall имя — завершить все процессы с именем
kill -9 PID — принудительное завершение
pkill имя — завершить процессы по имени

## Управление файлами

ls -la — подробный список файлов
ls -lh — размеры файлов в читаемом формате
find / -name "файл" — поиск файла
locate файл — быстрый поиск файла
which команда — найти путь к команде
whereis команда — найти команду и документацию

## Сетевые команды

ifconfig — информация о сетевых интерфейсах
ip addr — современная команда для IP
netstat -tulpn — активные соединения
ss -tulpn — альтернатива netstat
ping хост — проверить связь
traceroute хост — трассировка маршрута
nslookup домен — DNS запрос
dig домен — расширенный DNS запрос

## Управление пакетами

apt update — обновить список пакетов
apt upgrade — обновить установленные пакеты
apt install пакет — установить пакет
apt remove пакет — удалить пакет
apt search запрос — поиск пакетов
dpkg -l — список установленных пакетов
finger user показать информацию о пользователе

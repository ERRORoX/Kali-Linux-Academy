dmesg обнаруженное оборудование и сообщения загрузки

cat /proc/cpuinfo модель CPU

cat/proc/meminfo аппаратная память

cat/proc/interrupts список количества прерываний на CPU для каждого 1/0 устройства

Ishw отобразить информацию о конфигурации оборудования системы

Isblk отобразить информацию о блочных устройствах в Linux

free -m используемая и свободная память (-т для МБ)

Ispci -tv показать РСІ-устройства

Isusb -tv показать USB-устройства

dmidecode показать информацию об оборудовании из BIOS

hdparm -i /dev/sda показать информацию о диске sda

hdparm -tT /dev/sda провести тест скорости чтения диска sda

badblocks -s /dev/sda проверить на наличие нечитаемых блоков на диске sda
